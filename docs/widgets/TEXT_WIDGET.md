# Text Widget

Simple text entry widget for notes and content.

## Configuration

1. Add widget to dashboard
2. Click in widget to start typing
3. Text auto-saves as you type

## Features

- Real-time editing
- Auto-save
- Transparent background
- Theme-aware text color
- Plain text editor (textarea)

## Use Cases

- Quick notes
- Task lists
- Reminders
- Documentation snippets
- Formatted text display
