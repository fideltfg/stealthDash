# Image Widget

Display images from URLs with various fit modes.

## Configuration

1. Click to configure image in widget
2. Enter image URL
3. Choose object-fit mode:
   - **Contain**: Image fits within bounds (default)
   - **Cover**: Image fills widget, may crop
   - **Fill**: Stretch image to fill space
   - **None**: Display at original size
4. Optional: Add alt text for accessibility

## Supported Formats

- JPG/JPEG
- PNG
- GIF
- WebP
- SVG

## Example URLs

```
https://example.com/image.jpg
https://picsum.photos/400/300
```
