# Clock Widget

Display current time with timezone support.

## Configuration

1. Add widget
2. Select timezone from dropdown
3. Choose options:
   - **Format**: 12-hour or 24-hour
   - **Show Timezone**: Display timezone name below clock

## Display

- Current time (with seconds)
- Current date (full format: weekday, month, day, year)
- Timezone name (optional)
- Updates every second

## Customization

- Choose any IANA timezone
- 12-hour or 24-hour time format
- Toggle timezone name visibility
