# Environment Canada Widget

Canadian weather data from Environment Canada.

## Configuration

1. Add widget
2. Enter geographic coordinates:
   - **Latitude**: e.g., `43.65` (Toronto)
   - **Longitude**: e.g., `-79.38` (Toronto)
3. Choose language:
   - **English** (e)
   - **French** (f)
4. Set refresh interval (default: 300 seconds)

## Features

- Current conditions
- Weather forecast
- Weather warnings/alerts
- No API key required
- Direct RSS feed from Environment Canada

## Getting Coordinates

- Use Google Maps: Right-click location → "What's here?"
- Use [LatLong.net](https://www.latlong.net/)
- Environment Canada weather pages show coordinates

## Supported Regions

- All of Canada
- Automatic updates from official sources
