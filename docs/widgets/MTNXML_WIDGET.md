# MTN XML Widget

Display ski resort conditions from MTNXML feeds.

## Overview

The MTN XML widget is designed to parse and display ski resort data from MTNXML format feeds, showing lift status, trail conditions, snow reports, and weather.

## Configuration

1. Add widget
2. Enter MTNXML feed URL from a ski resort
3. Choose display options:
   - **Show Lifts**: Display lift status
   - **Show Trails**: Display trail conditions
   - **Show Snow**: Display snow reports
   - **Show Weather**: Display weather conditions
4. Set refresh interval
5. Choose display mode (Summary or Detailed)

## Features

- Real-time lift status (open/closed/on hold)
- Trail conditions and counts
- Snow depth and conditions
- Weather information
- Automatic refresh

## MTNXML Format

MTNXML is a standardized XML format used by ski resorts to share operational data. The widget parses this specific format to display resort information.

## Use Cases

- Monitor your favorite ski resort
- Track lift operations
- Check snow conditions
- Plan your ski day
