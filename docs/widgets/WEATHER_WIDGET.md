# Weather Widget

Display current weather and forecast using Open-Meteo API.

## Requirements

- Location name (city, country)
- No API key required!

## Setup

1. Add widget
2. Enter location:
   - Examples: `London`, `New York`, `Tokyo`, `Paris, France`
3. Widget automatically fetches weather data

## Features

- **No API Key Required**: Uses free Open-Meteo API
- Automatic location geocoding
- Current conditions with temperature
- "Feels like" temperature
- Weather description with icons
- Humidity percentage
- Wind speed
- 5-day forecast with:
  - Daily high/low temperatures
  - Weather icons
  - Precipitation probability

## Display

### Current Weather
- Location name and country
- Current temperature (°C)
- Weather icon and description
- Feels-like temperature
- Humidity and wind speed

### 5-Day Forecast
- Day of week
- Weather icon
- High/low temperatures
- Precipitation probability

## Data Source

Weather data provided by:
- **Geocoding**: Open-Meteo Geocoding API
- **Weather**: Open-Meteo Weather API
- All services are free and require no registration
